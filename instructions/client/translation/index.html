    Translation

    In order to do the translation go to folder res of common project located at: /common/src/main/res/ There is a file named strings.xml inside values folder copy the file and create a new values folder with your language id inside res folder then put this strings.xml file inside it. now edit this strings.xml file using any text editor of your choice. Now if users phone language is set to that language it will automatically choice that language for user.
